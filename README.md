# Anotações
Aplicativo desenvolvido em Ionic5 com Angular e Cordova de lista de compras

O aplicativo permite:
- Cadastrar itens com:
  - Nome;
- Deletar itens;
- Visualizar itens;

- Criar listas com:
  - Titulo
  - Data
  - Lista de Itens

- Remover itens da lista;
- Adicionar itens a lista;
- Editar lista;
- Deletar lista;
